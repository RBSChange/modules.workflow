Les différents utilisateurs possibles et leur rôle

Il est possible d’assigner des rôles aux utilisateurs de Change :

  • Le rédacteur : pourra créer de nouvelles pages ou de nouvelles versions d’une page, mais devra demander une validation avant de pouvoir les mettre en ligne sur un site.
  • Le valideur : pourra autoriser ou refuser la publication d’une nouvelle page ou d’une nouvelle version de page.

Dans ce cadre, les pages ou version de pages subiront un cycle de vie selon qu’elles ont été validées ou non.
