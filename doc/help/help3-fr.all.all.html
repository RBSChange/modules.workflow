Le processus de validation des pages

Le processus de validation nécessite aux valideurs d’effectuer les tâches de validation.

Le valideur est informé des tâches qu’il doit effectuer de plusieurs façons :

  • Par une alerte mail
  • Par le tableau de bord de suivi des processus de validation

Chaque contributeur dispose d'un tableau de bord de suivi des contenus pour lesquels il dispose de droits.

Ce tableau :

  • Liste, par type de contenu (module) les contenus créés, en attente de validation, validés, publiés, archivés (classement par type de document et dates).
  • Liste les tâches à effectuer : les documents en attente de validation.
  • Donne accès à une archive de toutes les actions menées pour un document.
